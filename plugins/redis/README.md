## Redis Client.

Currently supports jedis 2.4.x ~ 2.9.0

### Pinpoint Configuration
pinpoint.config

#### Set enable options.
~~~
# Enable/Disable
# Default value is true.
profiler.redis.enable=true

# IO(write/read) times.
# Default value is true.
profiler.redis.io=true

# Redis pipeline
# Default value is true.
profiler.redis.pipeline=true
~~~
